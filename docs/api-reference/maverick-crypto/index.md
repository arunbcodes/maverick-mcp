# maverick-crypto API Reference

Comprehensive cryptocurrency analysis package.

## Overview

`maverick-crypto` provides 35+ MCP tools across 7 categories:

| Category | Tools | Description |
|----------|-------|-------------|
| Data | 2 | Price and OHLCV data |
| Technical | 3 | RSI, MACD, support/resistance |
| CoinGecko | 6 | Trending, Fear/Greed, market data |
| Backtesting | 4 | Strategy testing and optimization |
| Portfolio | 5 | Mixed asset optimization |
| DeFi | 9 | TVL, yields, DEX pools |
| News | 6 | Aggregation and sentiment |
